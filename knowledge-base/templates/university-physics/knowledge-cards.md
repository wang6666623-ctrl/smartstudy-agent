## 第1章：质点运动学

### KP-001: 参考系与坐标系
- **定义**：描述物体运动时必须选定的参照物体（参考系），以及固定在参考系上的数学坐标框架（坐标系）
- **要点**：
  1. 运动描述的相对性：同一运动在不同参考系中描述不同
  2. 常用坐标系：直角坐标系(x,y,z)、极坐标系(r,θ)、自然坐标系(切向+法向)
  3. 参考系的选择原则：以简化问题为目标
  4. 惯性系与非惯性系：牛顿定律成立的参考系称为惯性系
- **难度**：简单
- **前置知识**：无
- **关联**：位置矢量(KP-002)、牛顿第一定律(KP-005)

### KP-002: 位置矢量与位移
- **定义**：位置矢量 r 描述质点在空间中的位置，位移 Δr = r(t+Δt) - r(t) 描述位置变化
- **要点**：
  1. 位矢 r 是时间的函数：r(t) = x(t)i + y(t)j + z(t)k
  2. 位移是矢量，与路径无关，只取决于初末位置
  3. 路程是标量，是实际轨迹的长度，一般 |Δr| ≤ Δs
  4. 在直角坐标系中位移分量：Δx, Δy, Δz
- **难度**：简单
- **前置知识**：KP-001
- **关联**：速度与加速度(KP-003)、运动方程(KP-004)

### KP-003: 速度与加速度
- **定义**：速度 v = dr/dt 是位移对时间的变化率；加速度 a = dv/dt 是速度对时间的变化率
- **要点**：
  1. 平均速度 = Δr/Δt，瞬时速度 = dr/dt
  2. 速率 |v| 是速度的大小（标量），平均速率 = 路程/时间
  3. 加速度分量：切向加速度 a_t = dv/dt（改变速率），法向加速度 a_n = v²/ρ（改变方向）
  4. 加速度是矢量，方向指向速度变化的"凹侧"
- **难度**：中等
- **前置知识**：KP-002
- **关联**：匀变速运动(KP-004)、圆周运动(KP-004)

### KP-004: 典型运动
- **定义**：匀变速直线运动、抛体运动、圆周运动三类基本运动模式
- **要点**：
  1. 匀变速直线运动：v = v₀ + at, x = x₀ + v₀t + ½at², v² - v₀² = 2a(x-x₀)
  2. 抛体运动：水平匀速+竖直匀变速的矢量叠加，轨迹为抛物线
  3. 圆周运动：ω = dθ/dt, v = rω, a_t = rα, a_n = rω² = v²/r
  4. 运动叠加原理：复杂运动可分解为独立方向的简单运动
- **难度**：中等
- **前置知识**：KP-003
- **关联**：牛顿第二定律(KP-006)

## 第2章：质点动力学

### KP-005: 牛顿第一定律（惯性定律）
- **定义**：任何物体在不受外力作用时，将保持静止或匀速直线运动状态
- **要点**：
  1. 定性表述：力是改变运动状态的原因，而非维持运动的原因
  2. 惯性：物体保持原有运动状态的性质，质量是惯性的量度
  3. 惯性参考系：牛顿第一定律成立的参考系
  4. 与亚里士多德观点的区别：力不是维持速度的原因
- **难度**：简单
- **前置知识**：KP-001, KP-004
- **关联**：牛顿第二定律(KP-006)、伽利略相对性原理(KP-049)

### KP-006: 牛顿第二定律
- **定义**：物体加速度与合外力成正比、与质量成反比，F = ma
- **要点**：
  1. 矢量形式：F = ma = m(dv/dt) = m(d²r/dt²)
  2. 分量形式：Fx = max, Fy = may, Fz = maz
  3. 自然坐标：Ft = m(dv/dt), Fn = m(v²/ρ)
  4. 力的叠加原理：合外力 = 各力的矢量和
  5. 解题步骤：选对象→隔离→受力分析→建坐标系→列方程→求解
- **难度**：中等
- **前置知识**：KP-005, KP-003
- **关联**：牛顿第三定律(KP-007)、常见力(KP-008)

### KP-007: 牛顿第三定律
- **定义**：两个物体之间的作用力 F₁₂ 和反作用力 F₂₁ 大小相等、方向相反，作用在同一直线上
- **要点**：
  1. F₁₂ = -F₂₁
  2. 作用力与反作用力分别作用在不同物体上，不能抵消
  3. 同时存在、同时消失、同种性质
  4. 与平衡力的区别：平衡力作用在同一物体上
- **难度**：简单
- **前置知识**：KP-006
- **关联**：动量守恒(KP-015)

### KP-008: 常见力与受力分析
- **定义**：重力、弹性力、摩擦力、万有引力等自然界基本力的性质和应用
- **要点**：
  1. 重力：G = mg，竖直向下，在地表附近近似匀强
  2. 弹性力（胡克定律）：F = -kx，方向指向平衡位置
  3. 摩擦力：静摩擦力 f_s ≤ μ_s·N，滑动摩擦力 f_k = μ_k·N
  4. 万有引力：F = Gm₁m₂/r²，G = 6.67×10⁻¹¹ N·m²/kg²
  5. 受力分析是解题的第一步：画隔离体图，逐个标出所有力
- **难度**：中等
- **前置知识**：KP-006
- **关联**：非惯性系(KP-009)、保守力(KP-011)

### KP-009: 非惯性系与惯性力
- **定义**：相对于惯性系做加速运动的参考系，在此系中需引入惯性力才能使牛顿定律成立
- **要点**：
  1. 平动加速系：惯性力 F_i = -ma₀（a₀为参考系加速度）
  2. 转动系：离心惯性力 F_cf = mω²r（沿径向向外）
  3. 科里奥利力：F_cor = -2m(ω × v')，垂直于速度和转轴
  4. 惯性力不是真实力（无施力物体），但在非惯性系中效果"真实"
- **难度**：困难
- **前置知识**：KP-006
- **关联**：刚体转动(KP-020)、圆周运动(KP-004)

## 第3章：功和能

### KP-010: 功与功率
- **定义**：功 dW = F·dr 是力在位移方向的分量与位移大小的乘积；功率 P = dW/dt 是做功的快慢
- **要点**：
  1. 恒力做功：W = F·Δr = FΔr·cosθ
  2. 变力做功：W = ∫F·dr（路径积分）
  3. 功是标量，可正可负（力与位移夹角 > 90° 时做负功）
  4. 功率：P = F·v
- **难度**：中等
- **前置知识**：KP-003, KP-006
- **关联**：动能定理(KP-011)、保守力(KP-012)

### KP-011: 动能定理
- **定义**：合外力对物体做的功等于物体动能的增量，W = ΔE_k = ½mv₂² - ½mv₁²
- **要点**：
  1. 动能 E_k = ½mv²
  2. 动能定理是标量方程，计算时比矢量方程简便
  3. 适用于单个质点，也适用于质点系（系统动能定理）
  4. 功是过程量，动能是状态量
- **难度**：中等
- **前置知识**：KP-010
- **关联**：功能原理(KP-013)、保守力(KP-012)

### KP-012: 保守力与势能
- **定义**：做功与路径无关只取决于初末位置的力称为保守力；势能是储存在保守力场中的能量
- **要点**：
  1. 保守力条件：∮F·dr = 0（沿闭合路径做功为零）或 ∇×F = 0
  2. 重力势能：E_p = mgh（取零势能面后）
  3. 弹性势能：E_p = ½kx²（取平衡位置为零势能点）
  4. 万有引力势能：E_p = -Gm₁m₂/r（取无穷远为零势能点）
  5. 保守力做功 = -ΔE_p
- **难度**：中等
- **前置知识**：KP-010, KP-008
- **关联**：机械能守恒(KP-013)

### KP-013: 功能原理与机械能守恒
- **定义**：外力做功和非保守内力做功之和等于系统机械能增量；只有保守力做功时机械能守恒
- **要点**：
  1. 功能原理：W_ext + W_nc_int = ΔE = ΔE_k + ΔE_p
  2. 机械能守恒条件：W_ext = 0 且 W_nc_int = 0（只有保守内力做功）
  3. E_k + E_p = 常量
  4. 解题步骤：选系统→分析力（区分保守/非保守）→判断是否守恒→列方程
  5. 用能量法解题常比动力学法更简洁
- **难度**：中等
- **前置知识**：KP-011, KP-012
- **关联**：动量守恒(KP-015)、碰撞(KP-018)

### KP-014: 质心运动定理
- **定义**：质点系质心的运动等效于全部质量集中在质心、全部外力作用在该点的质点运动
- **要点**：
  1. 质心位置：r_c = Σ(m_i·r_i)/M
  2. 质心运动定理：F_ext = M·a_c
  3. 内力不改变质心的运动状态
  4. 质心参考系（零动量参考系）：以质心为原点的参考系
- **难度**：中等
- **前置知识**：KP-006
- **关联**：动量定理(KP-015)、刚体运动(KP-019)

## 第4章：动量与角动量

### KP-015: 动量定理与动量守恒
- **定义**：合外力的冲量等于系统动量的增量；合外力为零时系统动量守恒
- **要点**：
  1. 动量：p = mv（矢量）
  2. 冲量-动量定理：∫F·dt = Δp（积分形式）
  3. 动量守恒条件：F_ext = 0（或某方向为零则该方向分量守恒）
  4. 守恒定律比牛顿定律更基本，适用范围更广
  5. 应用：火箭推进、碰撞问题
- **难度**：中等
- **前置知识**：KP-006, KP-007
- **关联**：碰撞(KP-018)、角动量(KP-017)

### KP-016: 变质量系统
- **定义**：质量随时间变化的系统的动力学方程
- **要点**：
  1. 密舍尔斯基方程：m(dv/dt) = F_ext + u(dm/dt)，其中u为喷射相对速度
  2. 火箭方程：v = u·ln(m₀/m)
  3. 适用场景：火箭发射、落链问题、雨滴下落
  4. 关键是正确写出动量变化的微分方程
- **难度**：困难
- **前置知识**：KP-015
- **关联**：动量守恒(KP-015)

### KP-017: 角动量定理
- **定义**：合外力矩等于角动量的变化率；合外力矩为零时角动量守恒
- **要点**：
  1. 角动量：L = r × p = r × mv（对点定义）
  2. 力矩：τ = r × F
  3. 角动量定理：τ = dL/dt
  4. 角动量守恒条件：合外力矩 τ_ext = 0
  5. 开普勒第二定律是角动量守恒的体现
- **难度**：中等
- **前置知识**：KP-015, KP-004
- **关联**：刚体转动(KP-020)、万有引力(KP-008)

### KP-018: 碰撞
- **定义**：两个或多个物体在极短时间内发生强烈相互作用的过程
- **要点**：
  1. 完全弹性碰撞：动能守恒 + 动量守恒，e = 1
  2. 完全非弹性碰撞：碰后粘在一起，动能损失最大，e = 0
  3. 非完全弹性碰撞：0 < e < 1，恢复系数 e = (v₂'-v₁')/(v₁-v₂)
  4. 对心碰撞（一维）：动量守恒 + 恢复系数方程解之
  5. 碰撞中的动能损失：ΔE_k = ½μ(v₁-v₂)²(1-e²)，μ为约化质量
- **难度**：中等
- **前置知识**：KP-015, KP-011, KP-013
- **关联**：角动量守恒(KP-017)

## 第5章：刚体力学

### KP-019: 刚体运动学
- **定义**：刚体（形状大小不变的物体）的平动和定轴转动
- **要点**：
  1. 刚体运动 = 平动 + 转动
  2. 平动：所有点位移相同，可用质心运动描述
  3. 定轴转动：角位移 θ(t), 角速度 ω = dθ/dt, 角加速度 α = dω/dt
  4. 线量与角量关系：v = rω, a_t = rα, a_n = rω²
  5. 自由度：空间一般运动6自由度（3平动+3转动）
- **难度**：中等
- **前置知识**：KP-004, KP-014
- **关联**：转动定律(KP-020)、转动惯量(KP-021)

### KP-020: 转动定律
- **定义**：刚体绕固定轴的转动满足 τ = Iα（力矩=转动惯量×角加速度）
- **要点**：
  1. τ = Iα 类比 F = ma，力矩类比力，转动惯量类比质量
  2. 力矩 τ = r × F，大小 τ = rFsinθ
  3. 转动定律是刚体转动的动力学基本方程
  4. 解题步骤：受力分析→计算力矩→计算I→列转动方程→求解
- **难度**：中等
- **前置知识**：KP-019, KP-017
- **关联**：转动惯量(KP-021)、角动量定理(KP-017)

### KP-021: 转动惯量
- **定义**：I = Σ(m_i·r_i²)，是刚体对转轴转动惯性的量度
- **要点**：
  1. 离散质点：I = Σm_i·r_i²
  2. 连续体：I = ∫r²·dm
  3. 平行轴定理：I = I_c + Md²（d为两平行轴距离）
  4. 正交轴定理（薄板）：I_z = I_x + I_y
  5. 常见形状：细杆端轴 I=⅓ML²、圆盘绕中心轴 I=½MR²、圆环绕中心轴 I=MR²、球体绕直径 I=⅖MR²
- **难度**：中等
- **前置知识**：KP-019, KP-020
- **关联**：转动动能(KP-022)

### KP-022: 刚体转动动能与功
- **定义**：刚体定轴转动的动能 E_k = ½Iω²，力矩做功 W = ∫τ·dθ
- **要点**：
  1. 转动动能：E_k = ½Iω²（类比 ½mv²）
  2. 力矩的功：W = ∫τ·dθ（类比 W = ∫F·dx）
  3. 刚体总动能 = 质心平动动能 + 绕质心转动动能：E_k = ½Mv_c² + ½I_cω²
  4. 动能定理（转动）：W = ΔE_k = ½Iω₂² - ½Iω₁²
- **难度**：中等
- **前置知识**：KP-020, KP-021, KP-011
- **关联**：机械能守恒(KP-013)、角动量守恒(KP-023)

### KP-023: 角动量守恒（刚体）
- **定义**：刚体定轴转动中，合外力矩为零时角动量 L = Iω 守恒
- **要点**：
  1. 刚体角动量：L = Iω
  2. 守恒条件：τ_ext = 0
  3. 角动量守恒时 Iω = 常量（如花样滑冰收缩手臂I↓→ω↑）
  4. 非刚体系统：L总 = 常量（I可变）
  5. 应用：陀螺仪、回转仪、进动现象
- **难度**：中等
- **前置知识**：KP-017, KP-021
- **关联**：转动定律(KP-020)、碰撞(KP-018)

## 第6章：流体力学

### KP-024: 流体静力学
- **定义**：静止流体的力学性质，包括压强分布和阿基米德原理
- **要点**：
  1. 静止流体内压强：p = p₀ + ρgh（随深度线性增大）
  2. 在同一水平面压强处处相等
  3. 帕斯卡原理：封闭流体中压强等值传递
  4. 阿基米德原理：浮力 = 排开流体的重力，F浮 = ρ_流gV排
  5. 表面张力：液体表面存在使表面积缩小的力，σ = F/l
- **难度**：中等
- **前置知识**：KP-006, KP-008
- **关联**：伯努利方程(KP-025)

### KP-025: 理想流体的伯努利方程
- **定义**：理想流体（不可压缩+无黏性）稳定流动时，p + ½ρv² + ρgh = 常量
- **要点**：
  1. 连续性方程（质量守恒）：S₁v₁ = S₂v₂（同一流管）
  2. 伯努利方程实质是流体中的机械能守恒
  3. 适用条件：理想流体、稳定流动、沿同一条流线
  4. 应用：流速计（皮托管）、流量计（文丘里管）、飞机升力解释、虹吸现象
- **难度**：中等
- **前置知识**：KP-013, KP-024
- **关联**：黏性流体(KP-026)

### KP-026: 黏性流体与雷诺数
- **定义**：实际流体中的黏性（内摩擦力）及其对流动的影响
- **要点**：
  1. 牛顿黏性定律：f = η·S·(dv/dy)（相邻流层间的内摩擦力）
  2. 雷诺数 Re = ρvL/η：判断层流(Re<2300)与湍流(Re>4000)
  3. 泊肃叶公式：圆管中黏性流体的流量 Q = πR⁴Δp/(8ηL)
  4. 斯托克斯定律：小球在黏性流体中的阻力 f = 6πηrv
- **难度**：困难
- **前置知识**：KP-025
- **关联**：伯努利方程(KP-025)

## 第7章：振动

### KP-027: 简谐振动
- **定义**：物体在平衡位置附近做 x = Acos(ωt + φ) 形式的周期性往复运动
- **要点**：
  1. 动力学方程：d²x/dt² + ω²x = 0
  2. 三个特征量：振幅A、角频率ω、初相位φ（由初始条件决定）
  3. 能量：E_k = ½mv² = ½kA²sin²(ωt+φ), E_p = ½kx² = ½kA²cos²(ωt+φ), E总 = ½kA²
  4. 弹簧振子 ω = √(k/m)，单摆（小角度）ω = √(g/l)，复摆 ω = √(mgd/I)
- **难度**：中等
- **前置知识**：KP-008, KP-013
- **关联**：阻尼振动(KP-028)、受迫振动(KP-029)

### KP-028: 阻尼振动
- **定义**：振动系统受阻力作用，振幅随时间衰减的振动
- **要点**：
  1. 方程：d²x/dt² + 2β·dx/dt + ω₀²x = 0（β为阻尼系数）
  2. 欠阻尼(β<ω₀)：振幅指数衰减 A=A₀e^{-βt}，仍有周期性
  3. 临界阻尼(β=ω₀)：最快回到平衡位置，无振荡
  4. 过阻尼(β>ω₀)：缓慢回到平衡位置，无振荡
  5. 品质因数 Q = ω₀/(2β) 衡量能量衰减快慢
- **难度**：中等
- **前置知识**：KP-027
- **关联**：受迫振动(KP-029)

### KP-029: 受迫振动与共振
- **定义**：在外加周期性驱动力作用下的振动
- **要点**：
  1. 稳态解频率 = 驱动力频率（与固有频率不同）

## 第8章：波动

### KP-031: 机械波的基本概念
- **定义**：振动在介质中的传播，传播的是振动状态（能量和动量），而非介质本身
- **要点**：
  1. 横波（质点振动⊥传播方向）vs 纵波（质点振动∥传播方向）
  2. 波速：v = λ/T = λf（λ波长、T周期、f频率）
  3. 波速取决于介质性质：弦线 v=√(T/μ)，固体纵波 v=√(Y/ρ)，气体声波 v=√(γRT/M)

### KP-032: 简谐波（平面简谐波）

### KP-033: 波的叠加与干涉

  3. 相长干涉：Δφ = 2kπ，A_max = A₁+A₂

### KP-034: 驻波

### KP-040: 理想气体模型与压强公式

### KP-042: 麦克斯韦速率分布

## 第11章：热力学基础

### KP-044: 热力学第一定律
- **定义**：系统内能的增量 = 吸收的热量 + 外界对系统做功，dU = δQ + δW（或dU = δQ - pdV）
- **要点**：
  1. 内能U是态函数（状态量），Q和W是过程量
  2. 常用形式：Q = ΔU + W = ΔU + ∫pdV
  3. 定容热容 C_V = (∂U/∂T)_V，定压热容 C_p = (∂H/∂T)_p（H=U+pV焓）
  4. 迈耶公式：C_p - C_V = R（理想气体）
  5. 绝热过程：pV^γ = 常数，γ = C_p/C_V
- **难度**：中等
- **前置知识**：KP-041, KP-010
- **关联**：热力学第二定律(KP-046)、卡诺循环(KP-047)

### KP-045: 热力学过程分析
- **定义**：四种基本准静态过程（等容、等压、等温、绝热）的热力学分析
- **要点**：
  1. 等容过程：W=0, Q_V = ΔU = νC_VΔT
  2. 等压过程：W = pΔV, Q_p = νC_pΔT, ΔU = νC_VΔT
  3. 等温过程：ΔU=0, Q = W = νRTln(V₂/V₁)（理想气体）
  4. 绝热过程：Q=0, W = -ΔU，pV^γ=常数，TV^{γ-1}=常数
  5. 多方过程：pV^n = 常数（n为多方指数）
- **难度**：中等
- **前置知识**：KP-044, KP-041
- **关联**：卡诺循环(KP-047)、熵(KP-048)

### KP-046: 热力学第二定律
- **定义**：自然过程具有方向性——热量不能自发从低温传向高温，不可能从单一热源吸热完全转化为功而不产生其他影响
- **要点**：
  1. 克劳修斯表述：热量不能自动从低温物体传到高温物体
  2. 开尔文-普朗克表述：不可能制成一种循环工作的热机，只从一个热源吸热完全变为功
  3. 两种表述等价（可用反证法证明）
  4. 可逆过程与不可逆过程：无耗散效应的准静态过程是可逆的
  5. 热力学第二定律是统计性规律（微观可逆、宏观不可逆）
- **难度**：中等
- **前置知识**：KP-044
- **关联**：卡诺循环(KP-047)、熵(KP-048)

### KP-047: 卡诺循环与热机效率
- **定义**：卡诺循环是工作于两个热源之间的理想可逆循环，由两个等温过程和两个绝热过程组成
- **要点**：
  1. 卡诺循环效率：η_C = 1 - T₂/T₁（仅与高低温热源温度有关）
  2. 卡诺定理：工作在两热源间的一切可逆热机效率相同=η_C，不可逆热机效率<η_C
  3. 制冷系数：ε = Q₂/W = T₂/(T₁-T₂)（从低温吸热/外界做功）
  4. 实际热机效率远低于卡诺效率（不可逆损失）
  5. T₁T₂均为绝对温度(K)
- **难度**：中等
- **前置知识**：KP-045, KP-046
- **关联**：熵(KP-048)

### KP-048: 熵与熵增原理
- **定义**：熵S是态函数，dS = δQ_rev/T；孤立系统熵永不减少
- **要点**：
  1. 克劳修斯不等式：∮δQ/T ≤ 0（等号对应可逆循环）
  2. 熵变计算：ΔS = ∫δQ_rev/T（沿可逆路径积分）

## 第12章：狭义相对论

### KP-049: 狭义相对论基本假设
- **定义**：爱因斯坦1905年提出的两个基本假设——相对性原理和光速不变原理
- **要点**：
  1. 相对性原理：所有惯性系中物理定律形式相同（不存在特殊参考系）
  2. 光速不变原理：真空中光速c在所有惯性系中相同，与光源运动无关，c≈3×10⁸ m/s
  3. 伽利略变换的失效：牛顿时空观在高速下不适用（电磁学规律不满足伽利略变换）
  4. 同时的相对性：不同地点发生的事件"同时"依赖于参考系
  5. 两个假设必然导致时空观的革命
- **难度**：中等
- **前置知识**：KP-005, KP-001
- **关联**：洛伦兹变换(KP-050)、时间膨胀(KP-051)

### KP-050: 洛伦兹变换
- **定义**：满足狭义相对论两个基本假设的时空坐标变换公式
- **要点**：
  1. 标准洛伦兹变换（S'沿+x方向相对S以v运动）：
     x' = γ(x - vt), y' = y, z' = z, t' = γ(t - vx/c²)
     其中 γ = 1/√(1-v²/c²)
  2. v≪c 时 γ≈1，回到伽利略变换
  3. 速度变换：u'_x = (u_x-v)/(1-vu_x/c²)，光速叠加仍为c
  4. 洛伦兹变换保证光速不变和物理定律协变性
  5. 闵可夫斯基时空：四维时空 x^μ = (ct, x, y, z)，时空间隔 Δs² 为洛伦兹不变量
- **难度**：困难
- **前置知识**：KP-049
- **关联**：时间膨胀(KP-051)、质能关系(KP-052)

### KP-051: 时间膨胀与长度收缩
- **定义**：运动时钟变慢（时间膨胀）和运动物体沿运动方向收缩的相对论效应
- **要点**：
  1. 时间膨胀：Δt = γ·Δτ（Δτ为本征时间/固有时，在随动系中测量）
  2. 长度收缩：L = L₀/γ（L₀为本征长度，在静止系中测量）
  3. 两种效应由洛伦兹变换直接导出
  4. μ子实验：高速μ子寿命显著延长（时间膨胀的实验验证）
  5. 孪生子佯谬：乘飞船旅行的兄弟更年轻（加速造成不对称）
- **难度**：中等
- **前置知识**：KP-050
- **关联**：质能关系(KP-052)

### KP-052: 相对论动力学与质能关系
- **定义**：高速运动物体的质量、动量和能量的相对论修正
- **要点**：
  1. 相对论质量：m = γ·m₀（m₀静质量，随速度增大而增大）
  2. 相对论动量：p = γ·m₀v
  3. 总能量：E = γ·m₀c² = mc²（质能关系）
  4. 静能：E₀ = m₀c²，动能：E_k = (γ-1)m₀c²
  5. 能量-动量关系：E² = (pc)² + (m₀c²)²
  6. 应用：核能释放（质量亏损→能量）、粒子加速器
- **难度**：中等

- **关联**：时间膨胀(KP-051)
