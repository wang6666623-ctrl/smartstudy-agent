## 模块一：力学（第1-6章）

### 第1章 质点运动学
```
参考系与坐标系
    ├── 位置矢量 r(t)
    ├── 位移 Δr（矢量，与路径无关）
    ├── 速度 v = dr/dt
    ├── 加速度 a = dv/dt
    │     ├── 切向加速度 a_t（改变速率）
    │     └── 法向加速度 a_n（改变方向）
    └── 典型运动
          ├── 匀变速直线运动
          ├── 抛体运动（水平匀速+竖直匀加速）
          └── 圆周运动（v=rω, a_n=v²/r）
```

### 第2章 质点动力学
```
牛顿运动定律
    ├── 第一定律：惯性定律 → 惯性系
    ├── 第二定律：F = ma
    │     ├── 矢量形式 + 分量形式
    │     └── 解题：隔离体法
    ├── 第三定律：F₁₂ = -F₂₁
    ├── 常见力
    │     ├── 重力 G=mg
    │     ├── 弹性力 F=-kx（胡克定律）
    │     ├── 摩擦力（静摩擦+滑动摩擦）
    │     └── 万有引力 F=Gm₁m₂/r²
    └── 非惯性系 → 惯性力
          ├── 平动加速系：F_i=-ma₀
          └── 转动系：离心力 + 科里奥利力
```

### 第3章 功和能
```
功与能
    ├── 功 dW = F·dr（过程量）
    ├── 动能 E_k = ½mv²（状态量）
    ├── 动能定理 W = ΔE_k
    ├── 保守力 → 势能
    │     ├── 重力势能 mgh
    │     ├── 弹性势能 ½kx²
    │     └── 万有引力势能 -GmM/r
    ├── 功能原理 W_ext+W_nc_int = ΔE
    ├── 机械能守恒 E_k+E_p = 常量
    └── 质心运动定理 F_ext = M a_c
```

### 第4章 动量与角动量
```
动量与角动量
    ├── 动量 p = mv
    ├── 冲量-动量定理 ∫Fdt = Δp
    ├── 动量守恒：F_ext=0 → p=常量
    ├── 变质量系统 → 火箭方程
    ├── 角动量 L = r × p
    ├── 角动量定理 τ = dL/dt
    ├── 角动量守恒：τ_ext=0 → L=常量
    └── 碰撞
          ├── 完全弹性（e=1）
          ├── 完全非弹性（e=0）
          └── 非完全弹性（0<e<1）
```

### 第5章 刚体力学
```
刚体运动 = 平动 + 转动
    ├── 运动学：θ, ω=dθ/dt, α=dω/dt
    ├── 转动定律 τ = Iα
    ├── 转动惯量 I = Σmr² = ∫r²dm
    │     ├── 平行轴定理 I=I_c+Md²
    │     └── 正交轴定理 I_z=I_x+I_y（薄板）
    ├── 转动动能 ½Iω²
    ├── 总动能 ½Mv_c² + ½I_cω²
    └── 角动量守恒 Iω = 常量
```

### 第6章 流体力学
```
流体力学
    ├── 流体静力学
    │     ├── p = p₀+ρgh
    │     ├── 帕斯卡原理
    │     ├── 阿基米德原理 F浮=ρgV排
    │     └── 表面张力
    ├── 理想流体
    │     ├── 连续性方程 S₁v₁=S₂v₂
    │     └── 伯努利方程 p+½ρv²+ρgh=常数
    └── 黏性流体
          ├── 雷诺数 Re=ρvL/η
          ├── 泊肃叶公式 Q∝R⁴
          └── 斯托克斯定律 f=6πηrv
```

## 模块二：振动与波（第7-8章）

### 第7章 振动
```
振动
    ├── 简谐振动 x=Acos(ωt+φ)
    │     ├── 弹簧振子 ω=√(k/m)
    │     ├── 单摆 ω=√(g/l)（小角度）
    │     └── 能量：E总=½kA²
    ├── 阻尼振动
    │     ├── 欠阻尼：振幅指数衰减
    │     ├── 临界阻尼：最快复原
    │     └── 过阻尼：缓慢复原
    ├── 受迫振动
    │     ├── 稳态→驱动力频率
    │     └── 共振：ω_d≈ω₀时振幅剧增
    └── 振动合成
          ├── 同方向同频率（矢量合成）
          ├── 拍现象 f_beat=|f₁-f₂|
          └── 李萨如图形（垂直合成）
```

### 第8章 波动
```
波动（振动在介质中的传播）
    ├── 基本概念
    │     ├── 横波 vs 纵波
    │     ├── v=λf（波速、波长、频率）
    │     └── 波动方程 ∂²y/∂x²=(1/v²)·∂²y/∂t²
    ├── 简谐波 y=Acos(ωt-kx+φ)
    │     ├── 能量密度 w̅=½ρA²ω²
    │     └── 波强 I=½ρA²ω²v
    ├── 叠加与干涉
    │     ├── 相干条件（同ω同方向恒Δφ）
    │     ├── 相长干涉 Δφ=2kπ
    │     └── 相消干涉 Δφ=(2k+1)π
    └── 驻波
          ├── y=2Acos(kx)cos(ωt)
          ├── 波节（不动点）间距λ/2
          ├── 本征频率 f_n=nv/(2L)
          └── 半波损失
```

## 模块三：光学（第9章）

### 第9章 光学
```
光学（光的波动性）
    ├── 几何光学
    │     ├── 反射定律
    │     ├── 折射定律 n₁sinθ₁=n₂sinθ₂
    │     ├── 全反射 θ_c=arcsin(n₂/n₁)
    │     └── 薄透镜 1/f=1/u+1/v
    ├── 光的干涉
    │     ├── 杨氏双缝 Δx=λD/d
    │     ├── 薄膜干涉 δ=2ndcosθ
    │     └── 等厚干涉（劈尖、牛顿环）
    ├── 光的偏振
    │     ├── 马吕斯定律 I=I₀cos²θ
    │     ├── 布儒斯特角 tanθ_B=n₂/n₁
    │     └── 双折射（o光+e光）
    ├── 光的衍射
    │     ├── 单缝衍射 asinθ=kλ
    │     ├── 圆孔衍射 θ_min=1.22λ/D
    │     └── 瑞利判据
    └── 光栅
          ├── dsinθ=kλ
          ├── 角色散 dθ/dλ
          └── 分辨本领 R=kN
```

## 模块四：热学（第10-11章）

### 第10章 气体动理论
```
气体动理论（微观→宏观）
    ├── 理想气体模型
    │     └── 压强公式 p=(2/3)n ε̅_k
    ├── 温度 (½mv²)平均 = (3/2)kT
    ├── 能均分定理 → 每自由度 ½kT
    │     ├── 单原子 (3/2)kT
    │     └── 双原子 (5/2)kT
    ├── 麦克斯韦速率分布
    │     ├── v_p < v̅ < v_rms
    │     └── T↑→分布展宽
    ├── 平均自由程 λ=1/(√2πd²n)
    └── 输运过程
          ├── 扩散（菲克定律）
          ├── 热传导（傅里叶定律）
          └── 黏性（牛顿定律）
```

### 第11章 热力学基础
```
热力学（宏观规律）
    ├── 第一定律 dU=δQ+δW
    │     ├── 内能U：态函数
    │     ├── 热量Q、功W：过程量
    │     └── C_p-C_V=R（理想气体）
    ├── 四种基本过程
    │     ├── 等容（W=0）
    │     ├── 等压（W=pΔV）
    │     ├── 等温（ΔU=0, Q=W）
    │     └── 绝热（Q=0, pV^γ=常数）
    ├── 第二定律
    │     ├── 克劳修斯表述
    │     ├── 开尔文表述
    │     └── 可逆与不可逆过程
    ├── 卡诺循环 η_C=1-T₂/T₁
    └── 熵
          ├── dS=δQ_rev/T
          ├── 熵增原理 ΔS≥0
          └── S=k·lnΩ（玻尔兹曼）
```

## 模块五：近代物理（第12章）

### 第12章 狭义相对论
```
狭义相对论
    ├── 两个基本假设
    │     ├── 相对性原理
    │     └── 光速不变原理
    ├── 洛伦兹变换
    │     ├── x'=γ(x-vt), t'=γ(t-vx/c²)
    │     └── γ=1/√(1-v²/c²)
    ├── 相对论运动学
    │     ├── 时间膨胀 Δt=γ·Δτ
    │     └── 长度收缩 L=L₀/γ
    └── 相对论动力学
          ├── m=γ·m₀
          ├── E=mc²（质能关系）
          ├── E²=(pc)²+(m₀c²)²
          └── E₀=m₀c²（静能）
```
