# 📖 通用学习方法论 — 知识卡片

---

### GP-001: 费曼学习法
- **定义**：通过"教给别人"来检验和加深自己理解的学习方法
- **要点**：
  1. 选择概念 → 尝试教给一个外行 → 发现解释不清的地方 → 回头学习 → 简化类比
  2. 核心原理：如果你不能简单地解释，说明你还没真正理解
  3. 适用场景：概念性知识的深度理解
  4. 自我检验：用最简单的语言重新表述知识点

### GP-002: 间隔重复 (Spaced Repetition)
- **定义**：在逐渐增大的时间间隔上复习知识，以对抗遗忘曲线
- **要点**：
  1. 艾宾浩斯遗忘曲线：学后1小时遗忘56%，1天后遗忘74%
  2. 最佳复习间隔：1天 → 3天 → 7天 → 14天 → 30天
  3. 工具：Anki、Supermemo
  4. 适用：记忆性内容（公式、定义、外语词汇）

### GP-003: 主动回忆 (Active Recall)
- **定义**：通过主动从记忆中检索信息来强化记忆，而非被动重读
- **要点**：
  1. 做法：合上书，尝试回忆刚才学的内容
  2. 比被动重读效率高50%以上
  3. 具体方法：自问自答、默写、做思维导图
  4. 与间隔重复结合效果最佳

### GP-004: 番茄工作法 (Pomodoro)
- **定义**：25分钟专注工作 + 5分钟休息的循环工作模式
- **要点**：
  1. 标准循环：25min专注 → 5min休息 → 重复4次 → 15-30min长休息
  2. 核心原则：单任务、无干扰、定时强制休息
  3. 防止疲劳积累，维持高专注力
  4. 可调整：根据个人情况调整时长（如50min/10min）

### GP-005: 时间管理四象限
- **定义**：按紧急度和重要度将任务分为四个象限的管理方法
- **要点**：
  1. 第一象限：紧急+重要 → 立即做
  2. 第二象限：不紧急+重要 → 计划做（最值得投入）
  3. 第三象限：紧急+不重要 → 委托/简化
  4. 第四象限：不紧急+不重要 → 减少/消除
  5. 关键：多投入第二象限，减少救火式工作

### GP-006: SMART目标法
- **定义**：制定具体、可衡量、可实现、相关、有时限的目标
- **要点**：
  1. Specific（具体的）：而不是模糊的"好好学习"
  2. Measurable（可衡量的）：量化指标
  3. Achievable（可实现的）：不过高也不过低
  4. Relevant（相关的）：与长期目标一致
  5. Time-bound（有时限的）：明确的截止日期

### GP-007: 康奈尔笔记法
- **定义**：将笔记页面分为笔记区、线索区和总结区的结构化笔记方法
- **要点**：
  1. 笔记区（右）：记录课堂/阅读内容
  2. 线索区（左）：提炼关键词、问题
  3. 总结区（底）：用自己的话总结一页内容
  4. 复习时：遮住笔记区，根据线索区回忆内容
  5. 本质是内置了主动回忆的结构

### GP-008: 心流状态与深度学习
- **定义**：完全沉浸在当前任务中的心理状态，是最高效的学习状态
- **要点**：
  1. 进入条件：挑战与技能匹配、目标清晰、即时反馈、无干扰
  2. 特征：时间感消失、高度专注、内在满足感
  3. 建议：固定时间地点学习、关闭所有通知、使用番茄钟
  4. 深度工作 > 浅层工作：1小时深度 > 3小时碎片
