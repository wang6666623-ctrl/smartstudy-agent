**🔬 论文核心贡献**

1. **提出 Transformer 架构**：完全基于注意力机制，摒弃了传统 RNN/CNN 的序列化计算方式，实现完全并行化的序列到序列学习。

2. **自注意力机制（Self-Attention）**：通过 Scaled Dot-Product Attention 计算序列中每个位置与其他所有位置的相关性，公式为 Attention(Q,K,V) = softmax(QK^T/√d_k)V。

3. **多头注意力（Multi-Head Attention）**：将 Q、K、V 线性投影到 h 个不同的子空间，并行执行注意力计算后拼接，使模型能够关注来自不同表示子空间的信息。

4. **位置编码（Positional Encoding）**：由于没有 RNN 的时序结构，使用正弦/余弦函数编码位置信息，使模型能感知序列顺序。

**🏗️ Transformer 架构组件**

```
输入序列
    │
    ▼
┌──────────────┐
│  位置编码     │ ← 正弦/余弦 (PE(pos,2i)=sin(pos/10000^(2i/d)))
├──────────────┤
│  多头自注意力  │ ← h=8 头, d_k=d_v=64
│  + Add & Norm │
├──────────────┤
│  前馈网络     │ ← FFN(x)=max(0, xW_1+b_1)W_2+b_2
│  + Add & Norm │
├──────────────┤
│     ×N       │ ← 编码器层数 N=6
└──────────────┘
    │
    ▼  (编码器输出 → 解码器交叉注意力)
```

**📖 全文摘要（498字）**

本文提出了 Transformer——一种全新的序列到序列模型架构，完全摆脱了循环神经网络和卷积神经网络，仅依赖注意力机制实现序列建模。Transformer 的核心创新在于自注意力机制：对输入序列中的每个位置，通过计算其与所有位置的相似度（QK^T）来加权聚合信息（V），并使用缩放因子 √d_k 防止点积值过大导致的梯度消失。为增强模型的表达能力，引入了多头注意力——将查询、键、值投影到 8 个不同的表示子空间，并行执行注意力计算后拼接。为弥补去除 RNN 后丢失的位置信息，使用正弦/余弦函数进行位置编码。

在机器翻译任务上的实验表明：Transformer 不仅在 WMT 2014 英德（28.4 BLEU）和英法（41.0 BLEU）翻译任务上达到了当时最优水平，而且训练速度显著快于基于 RNN 的模型——在 8 块 P100 GPU 上仅需 12 小时即可完成训练（Base 模型）。此外，该架构展现出良好的泛化能力，其核心组件（自注意力、位置编码、层归一化）随后被广泛应用于 BERT、GPT、ViT 等众多后续工作，成为现代 NLP 和深度学习的基础架构。截至 2026 年，论文引用量已超过 10 万次。
