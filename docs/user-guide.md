# 智学助手 — 用户指南

## 快速开始

### 通过 Claude Code 使用（推荐）

1. 确保已安装 Claude Code
2. 将项目文件置于工作目录
3. 启动智学助手：
   ```
   用 study-assistant 帮我...
   ```
   或者直接对话，智学助手会自动识别意图并调用对应 Skill。

### 通过 Web 界面使用

1. 打开 `web/index.html`（可用浏览器直接打开）
2. 在对话框中输入需求
3. 或点击上方的快捷功能按钮

## 五大功能使用指南

### 📖 资料速读
**场景**：你有一份课件/论文需要快速了解重点
**使用方式**：
```
帮我看一下这个课件：C:\courses\ai-chapter3.pptx
```
或者直接拖拽文件到对话框。

**产出**：知识卡片、层次化摘要、思维导图大纲

### ❓ 智能答疑
**场景**：学习中遇到不理解的概念
**使用方式**：
```
什么是A*搜索算法？能给我详细解释一下吗？
```
**产出**：三层递进式解答（简答→详解→延伸）

### 📅 复习规划
**场景**：临近考试需要科学备考
**使用方式**：
```
帮我制定人工智能导论的复习计划，7月22日考试，每天能学2小时
```
**产出**：四阶段按天复习计划表

### 📝 笔记整理
**场景**：课堂笔记零散需要整理
**使用方式**：
```
帮我整理以下笔记：
[粘贴零散笔记内容]
```
**产出**：结构化Markdown笔记 + Anki闪卡

### ⏰ DDL管理
**场景**：多门课程作业和考试需要管理
**使用方式**：
```
帮我记录DDL：AI导论实验报告7月10日截止，数据结构作业7月12日
```
**产出**：DDL时间线 + 优先级排序 + 冲突检测

## 知识库使用

智学助手内置了《人工智能导论》和通用学习方法论的知识库。你可以：

1. 直接提问课程相关问题，系统会自动检索知识库
2. 浏览知识库查看全部知识点
3. 知识库未覆盖的内容会使用通用知识回答并标注

## 技巧与建议

- 🎯 **具体提问**：越具体的问题得到的回答越精准
- 📚 **多轮对话**：智学助手会记住上下文，可以连续追问
- 📎 **多用文件**：直接上传课件比描述更高效
- 🔄 **定期更新**：告诉智学助手你的学习进展，它会调整计划
