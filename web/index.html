ℹ️ 关于智学助手

项目背景

智学助手是《人工智能导论》课程设计项目，选择"学习辅助与日常效率类 Agent/Skill"方向，以《大学物理（上）》为示范课程构建知识库，面向大学生打造的多功能AI学习伴侣。

技术架构

LLM 基座Claude API (Fable 5)
智能体框架Claude Code Agent + Skill
知识库Markdown + JSON 结构化存储
记忆模块JSON 文件 + 对话历史
Web 界面纯静态 HTML/CSS/JS

设计特征

  • 🧠 自主感知：接收文件/问题/日期，自动理解意图
  • 🔧 工具调用：搜索、文件处理、知识库检索、日历管理
  • 📋 规划决策：根据目标自动拆解任务并执行
  • 💾 记忆能力：学习进度、薄弱点、历史对话持久化
  • 🗣️ 自然交互：对话式交互，支持Web和CLI
