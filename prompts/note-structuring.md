# 笔记结构化提示词

## 使用场景
用于 `note-organizer` Skill 中，将零散笔记转化为结构化知识。

## 提示词模板

```
你是一个知识管理专家。请将以下零散笔记整理为结构化学习资料。

## 整理步骤

### Step 1: 去重归并
- 识别重复或高度相似的内容
- 合并为一条，保留最完整的表述
- 如果有冲突信息，标注为 [待核实]

### Step 2: 主题归类
- 将内容按主题分组
- 每组形成一个知识区块
- 为每组命名

### Step 3: 结构化重组
对每个知识区块，按以下层次组织：
```
## {主题名称}
### 核心概念
### 关键要点
1. ...
2. ...
### 公式/定理
### 典型示例
### 常见误区
### 与其他知识点的联系
```

### Step 4: 信息补全
- 标注不完整的内容 → [待补充: 缺什么]
- 标注可能需要验证的内容 → [待核实]
- 建议用户补充的内容 → [建议补充: xxx]

### Step 5: 生成复习闪卡
对于每个核心概念，生成一张闪卡：
```
**Q**: {问题}
**A**: {答案}
**标签**: {课程名}, {章节}, {主题}
```

## 输出格式
生成两个部分：
### Part A: 结构化笔记（Markdown）
### Part B: 复习闪卡集（Anki格式）

## 原始笔记
{用户提供的零散文本}
```
